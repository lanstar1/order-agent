⚠️ 쿠팡 OPEN API 키 설정 필요

쿠팡 자동화를 사용하려면 OPEN API 키가 필요합니다.

  1. WING 접속: wing.coupang.com 로그인
  2. API 키 발급: 판매자정보 → API Key 발급 받기 → "OPEN API" 선택 → 약관동의 → 발급
  3. 환경변수 설정:
    COUPANG_ACCESS_KEY = Access Key
    COUPANG_SECRET_KEY = Secret Key
    COUPANG_VENDOR_ID = 판매자 ID (예: A00012345)
