⚠️ ESM API 키 설정 필요

지마켓/옥션 자동화를 사용하려면 ESM Trading API 키가 필요합니다.

  1. API 키 신청: etapihelp@gmail.com으로 메일 발송
    내용: ESM 마스터 ID, 서비스 URL, 매출 규모
  2. 발급 후 환경변수 설정:
    ESM_MASTER_ID = ESM 마스터 ID
    ESM_SECRET_KEY = 발급받은 Secret Key
    ESM_SELLER_ID_G = 지마켓 판매자 ID
  3. ERP 거래처코드 설정:
    GMARKET_CUST_CODE = 지마켓 거래처코드 (이카운트)
